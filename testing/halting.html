Unileq Architecture

Given N All addresses and values are indexed mod N. Addresses are indexed from 0,1,...,N-1. Each address can have value 0,1,...,N-1. The instruction pointer can have value 0,1,...,N-1. The program halts if instruction "-1 X 0" is encountered, for any X. Given the N memory cells, with N possible values, and N values for the IP, there are N^N possible initial programs since IP=0, with N^(N+1) possible states. 0 - 1 1 - 1 2 - 4 3 - 27 4 - 256 5 - 3125 6 - 46656 7 - 823543 8 - 16777216 9 - 387420489 use an array to count predecessors make a list of predecessors trace back from halting state consider program that starts with x x 0 then all programs that start with this will never halt N*N^(N-3)/(N^N)=N^(N-2)/(N^N)=1/(N^2) all programs starting with -1 X 0 will halt this places an upper and lower bound on halting probability.
